# tutu-bookkeeping 
## 项目介绍
- 涂涂记账本是一个基于vue实现的项目，你可以通过记账页面记录每天的收支，然后在统计页面看到收支的整体情况，有记录显示，也有图表显示。
- 技术栈： vue, typescript, echarts, scss, webpack
